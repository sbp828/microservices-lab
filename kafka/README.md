# Kafka Event Flow Microservices

## Overview
This project demonstrates a Kafka producer-consumer flow using Spring Boot microservices.

## Structure
```
kafka/
├── producer-service/
├── consumer-service/
├── docker-compose.yml
└── README.md
```

## Features
- Producer sends messages to Kafka topic.
- Consumer listens and consumes messages.
- Docker Compose setup for Kafka & Zookeeper.

## Tech Stack
- Java 17
- Spring Boot
- Spring Kafka
- Maven
- Docker (Kafka & Zookeeper)
